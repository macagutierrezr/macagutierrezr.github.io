03 Mecanizado CNC - Respaldo

Mecanizado CNC Respaldo Silla Fresia

Para el mecanizado de el respaldo de la silla fresia se debe trabajar con corte 2.5D, ya que solo se necesitan cortes de profundidad pero de manera regular. Por un lado el trabajo de 2,5D se llama así porque solo permite una única elevación (z) el valor de cualquier superficie en un punto dado. Por otro lado el mecanizado se logra controlar tres ejes de manera simultánea, se combinan las habilidades de la impresión 3D con las del fresado.

Encargo

Utilizar Fusion 360 para crear simular el mecanizado del respaldo de la silla fresia.

Proceso mecanizado con imágenes y videos

Estrategias de corte 2,5D que se utilizarán:

1) Adaptative

2) Parallel

Para realizar el respaldo, hay que hacer una caja, simulando la madera, de manera que el respaldo quede adentro por completo. Para el mecanizado se usará la misma fresa que se usó para las otras partes de la silla, Flat and mill de 6mm de diámetro. Al igual que el otro mecanizado es importante definir bien la orientación de la fresa. Para hacer los cortes hay que hacer un sketch con project, donde se proyecte la zona que se cortará. Para hacerlo hay que hacer un sketch, seleccionar la cara, y seleccionar project, escogiendo el respaldo como proyeccción. Así se crea una proycección en la cual se puede empezar a mecanizar.

Para lograr los cortes hay que hacer 2 pasos:

  1. Adaptative: Es un mecanizado en donde se hace una limpieza o desbaste adaptativo a la madera, donde se sacará material para empezar el proceso de corte.

  1. Parallel: El paralell se trata de un mecanizado paralelo enl cual deja un acabado más perfecto. Para este proceso se utiliza la misma fresa pero Ball and mill, ya que se necesita que sea de punta redonda para un mejor acabado. Se deben hacer 2 parallel, por un lado y luego en passes de pone a 90ª y con stepover de 2 para una mejor resolución.
