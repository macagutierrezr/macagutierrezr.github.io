BONUS Grasshopper

Archivo grasshopper mecanizado

Encargo

Realizar el mecanizado de un archivo grasshopper en Fusion 360.

Proceso mecanizado con imágenes y videos

Para realizar el mecanizado se utiliza adaptative y parallel, al igual que el respaldo, pero con la fresa corta tal como se usó para la silla. En el caso del parallel se utiliza la fresa de punta redonda.

Para terminar se debe hacer un contour 2D agregándole tabs.
